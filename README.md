# Cyclistic Bike Analysis

This project analyzes Cyclistic bike-share data using SQLite, DBeaver, Excel, and visualization tools.

## Tools Used
- SQLite
- DBeaver
- Excel
- PowerPoint

## Project Files
- `cyclistic_process_phase.sql` — SQL queries for data cleaning and processing
- `cyclistic_analyze_phase.sql` — SQL queries for analysis
- `cyclistic_analyze_phase.xlsx` — analysis output file
- `Visualizations.xlsx` — charts and visual outputs
- `project_charts.pptx` — presentation with project charts
- `Report.pdf` — final project report
- `raw_data_sample.xlsx` — sample of raw data used in the project

## Project Goal
The goal of this project is to analyze user behavior patterns in Cyclistic bike-share data and generate useful business insights.

## Note
Due to file size limitations, a sample of the raw dataset is included in this repository instead of the full raw data files.
